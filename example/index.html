Dart Springel Library

Creates elements that clinging to the cursor, follow him at a certain distance.

Use constructor without limit

        new SpringElements(
            elements: '.nav a'
        );

Use constructor with limit

        new SpringElements(
            elements: '.mainLinks a',
            limit: 130
        );

Contain element must have *position: relative* in style

    HTML code:

    <div class="wrap">
        <a href="#"></a>
    <div>


    CSS code:

    .wrap {
        position: relative;
        display: block;
        float: left;

        width: 100%;
        height: 60px;

        padding: 10px;
    }

    .wrap a {
        background: #F14472;
        display: block;

        width: 60px;
        height: 60px;

        float: left;
        margin-right: 10px;
    }


    Dart code:

    new SpringElements(
        elements: 'wrap a',
        limit: 130
    );
